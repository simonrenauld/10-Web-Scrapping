============
Installation
============


Stable release
--------------

To install find_job_titles, run this command in your terminal:

.. code-block:: console

    $ pip install find_job_titles

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for find_job_titles can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/codinguncut/find_job_titles

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/codinguncut/find_job_titles/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install -e .


.. _Github repo: https://github.com/codinguncut/find_job_titles
.. _tarball: https://github.com/codinguncut/find_job_titles/tarball/master
